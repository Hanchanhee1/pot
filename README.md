# 🚀 한찬희 포트폴리오

신입 백엔드 개발자 한찬희의 포트폴리오 웹사이트입니다.

## 👨‍💻 소개

Spring Boot로 RESTful API를 설계하고, 데이터베이스를 모델링하며, AI 기술을 웹 서비스에 통합합니다.
클라우드 인프라를 활용한 배포와 운영 경험도 갖추고 있습니다.

**16개월간의 교육 과정**과 **4개의 프로젝트**를 통해 웹 개발의 기초를 탄탄히 다졌습니다.
JSL인재개발원에서 Spring 기반 웹 개발을, 스마트인재개발원에서 AI 기술 연동을 학습했으며,
**NAVER Cloud Platform Certified Professional 자격증**을 취득하여 클라우드 인프라에 대한 이해를 넓혔습니다.
일본 IT 기업에서의 해외 근무 경험도 있습니다.
